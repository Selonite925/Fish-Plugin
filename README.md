# Fish-plugin

适用于 TRSS-Yunzai / Yunzai-Bot 的独立钓鱼插件。

这个插件提供一套完整的群聊钓鱼玩法：每日钓鱼、鱼缸收藏、鱼市售鱼、鱼币商店、鱼竿鱼饵、传说鱼炼竿、每日鱼讯、钓鱼成就和图片面板。

## 功能概览

- `#钓鱼`：每日抛竿钓鱼，有概率钓到不同稀有度的鱼。
- `#钓鱼极速版`：一次结算当前所有可用钓鱼次数，跳过演出并用图片汇总结果。
- `#查看鱼缸`：查看收藏的鱼，支持升级鱼缸容量。
- `#鱼市`：出售鱼获换鱼币，购买鱼饵、鱼竿和额外钓鱼券。
- `#鱼竿 / #换竿`：切换不同鱼竿，影响等待时间、上鱼率、失败保护和稀有鱼倾向。
- `#鱼饵 / #换饵`：切换不同鱼饵，鱼饵会在钓鱼时消耗。
- `#炼竿`：消耗鱼缸里的 `legendary` 鱼，炼成特殊鱼竿。
- `#钓鱼祈愿`：100 鱼币祈愿 1 次，可获得祈愿限定鱼竿、鱼饵和免费祈愿。
- `#限时鱼讯`：每天刷新高活跃鱼，钓中可获得额外鱼币。
- `#钓鱼成就`：查看成就和奖励。
- `#钓鱼管理`：查看管理、维护和主人命令。

## 安装

进入你的 Yunzai-Bot 或 TRSS-Yunzai 根目录，然后执行：

```bash
git clone --depth=1 https://github.com/Selonite925/Fish-Plugin.git ./plugins/Fish-plugin
```

拉取完成后重启机器人。

如果你的环境使用 pnpm 管理依赖，通常不需要额外安装依赖；插件会使用 Yunzai 已有的运行环境和 puppeteer 图片渲染能力。

### 可选的时间增强依赖

插件默认不依赖额外时间库，没装也能正常运行；当前会优先使用原生 `Date`、`Intl.DateTimeFormat` 和框架自带定时任务。

如果你后面想给时间逻辑留扩展位，比如：

- 更统一的时区处理
- 更方便的时间格式化
- 后续扩展相对时间、时间区间、活动时段判断

可以额外安装可选依赖 `luxon`。Fish 插件已经内置了自动探测逻辑：

- 装了 `luxon`：自动启用增强时间处理
- 没装 `luxon`：自动回退到当前原生实现

可选安装方式：

```bash
cd ./plugins/Fish-plugin
pnpm install
```

或者只安装这个时间库：

```bash
pnpm add luxon
```

说明：

- 插件目录里的 `package.json` 已把 `luxon` 声明为 `optionalDependencies`
- 不安装不会报错，也不会影响现有日切、鱼讯、等待时间和时间戳记录

## 更新

进入机器人根目录，执行：

```bash
git -C ./plugins/Fish-plugin pull
```

更新后重启机器人。

如果服务器直连 GitHub 不稳定，主人可以在机器人内为 `#钓鱼更新` 设置代理：

```text
#钓鱼更新代理
#钓鱼更新代理 https://gh-proxy.com/
#钓鱼更新代理 https://gh-proxy.com/{url}
#钓鱼更新代理关闭
```

未设置代理时仍然使用 GitHub 直连；设置后 `#钓鱼更新` 的 fetch 和安全重装都会通过代理地址拉取。

## 卸载

停止机器人后删除插件目录：

```bash
rm -rf ./plugins/Fish-plugin
```

Windows PowerShell 可用：

```powershell
Remove-Item -Recurse -Force .\plugins\Fish-plugin
```

### 鱼市和售鱼

| 命令 | 说明 |
| --- | --- |
| `#鱼市` | 查看商店、鱼饵、鱼竿和售鱼提示 |
| `#售鱼 1` | 卖出 `#今日鱼获` 里显示的第 1 条 |
| `#售鱼 common` | 卖出今日所有 common 鱼 |
| `#售鱼 common 3` | 卖出今日 common 列表第 3 条 |
| `#售鱼 鱼缸3` | 卖出鱼缸展示列表第 3 条 |
| `#售鱼 全部` | 卖出今日所有可售鱼 |

`legendary` 和彩蛋鱼 `？` 不能出售。

`#售鱼 1` 这类“今日鱼获序号”按 `#今日鱼获` 展示出来的编号来认，不会再额外重排。

售鱼价格以不可售卖的 `legendary` 和彩蛋鱼为基准：二者合计上鱼概率为 2%，统一视作 1500 价值，基准期望为 30 鱼币/次成功上鱼。可售鱼按稀有度概率反推价值区间，当前区间为：`common` 35-80、`uncommon` 75-165、`rare` 150-330、`epic` 280-620。单条鱼价格会根据该鱼在本鱼种长度、重量范围内的位置浮动，重量占 70%、长度占 30%，最终始终限制在对应稀有度区间内。

### 买东西

| 命令 | 说明 |
| --- | --- |
| `#鱼市购买 鱼饵1` | 购买鱼市里的第 1 种鱼饵 |
| `#鱼市购买 鱼饵1*5` | 批量购买 5 包鱼饵 |
| `#鱼市购买 鱼饵1 5包` | 同样是批量购买 5 包鱼饵 |
| `#鱼市购买 鱼竿1` | 购买鱼市里的第 1 根鱼竿 |
| `#鱼市购买 额外钓鱼券` | 购买额外钓鱼次数 |
| `#鱼市购买 自定义鱼饵 桂花酒糟` | 用文本定制一份鱼饵 |

鱼饵价格按“包”计算；每包会提供 `packSize` 次库存，例如 1 包 3 份的鱼饵可以在钓鱼时消耗 3 次。

额外钓鱼券只在基础次数用完后消耗，并单独记录为额外竿；鱼缸升级后新增的基础次数仍会按基础竿补给玩家。

| 命令 | 说明 |
| --- | --- |
| `#炼竿预览 3` | 预览鱼缸展示列表第 3 条鱼会炼成什么鱼竿 |

### 钓鱼祈愿

| 命令 | 说明 |
| --- | --- |
| `#钓鱼祈愿` | 花费 100 鱼币祈愿 1 次 |
| `#钓鱼祈愿10` | 连续祈愿 10 次，单次命令最多 10 连 |
| `#钓鱼十连` | 同样是 10 连写法 |
| `#钓鱼祈愿清单` | 查看祈愿清单、金线进度和普通愿品期望 |

祈愿普通愿品长期回报约为 67% 左右，期望计算不计入限定愿品价值。当前限定愿品接口挂载 `金满而谦虚之竿`，估值按 2500 鱼币计算，回收价值为估值的一半。玩家获得当前限定愿品后，界面会提示“当前限定愿品已获得”，后续祈愿只走普通愿品清单。

祈愿获得的鱼饵会明确区分“包”和“份”：祈愿限定鱼饵按包展示，实际进入背包后按份消耗。例如 `渊金鱼饵 1包` 是 1 包 1 份，通常可钓 1 次；`谦光素饵 1包` 和 `镜水留饵 1包` 是 1 包 2 份，通常可钓 2 次。`再祈一愿` 按 1 次免费祈愿展示，面值视为 100 鱼币。

`金满而谦虚之竿` 是祈愿限定鱼竿，不能在鱼市直接购买。它可以用 `#金谦指定 鱼名` 或 `#金谦目标 rare 金龙鱼` 指定目标鱼：不同目标稀有度会改变本竿的整体水口，模拟期望按钓上来的鱼本身价值计算，不把命中目标的额外鱼币奖励计入平均每杆期望。钓到目标稀有度时，目标鱼在同稀有度里的出现感会大大增加，也会有同稀有度鱼影意外闯入；鱼种越多的稀有度，闯入鱼影越多，common 最多，legendary / 彩蛋较少。命中真正目标会给鱼币奖励，但其它额外鱼币收益不会叠加；每天最多调整 1 次目标，重复指定同一目标可刷新闯入鱼影。

### 鱼缸升级

鱼缸每升 1 级增加 5 格容量，并让玩家本人每日基础钓数永久 +2。升级消耗会随等级递增：升到第 N 级需要 N 条 `legendary` 鱼，或 3N 条 `epic` 鱼。

## 玩法说明

### 稀有度

鱼的稀有度从低到高大致为：

```text
common -> uncommon -> rare -> epic -> legendary -> ？
```

`？` 是彩蛋稀有度，出现率非常低。

### 鱼竿怎么选

- 想钓得快：优先买 `疾风短竿`、`柳影细竿`。
- 想少空竿：优先买 `稳钓重竿`。
- 想追稀有鱼：优先买 `猎珍长竿`。
- 想守大鱼：优先买 `阔鳞量竿`、`坠岩守竿`。

### 鱼饵怎么选

- 想稳定上鱼：买 `香谷鱼饵`。
- 想多钓普通鱼攒鱼币：买 `银鳞鱼饵` 或 `青苔鱼饵`。
- 想追稀有鱼：买 `沉流鱼饵`、`乌贼鱼饵`、`雷息鱼饵`。
- 想玩随机效果：买 `自定义鱼饵`。

## 数据文件

插件数据主要保存在：

| 文件 | 说明 |
| --- | --- |
| `fishdata/fishData.json` | 用户主数据、鱼缸、鱼币、鱼竿、鱼饵、成就 |
| `fishdata/fishConfig.json` | 基础配置、封竿群 |
| `fishdata/baitData.json` | 手动打窝数据 |
| `fishdata/lostItems.json` | 失败事件掉落物 |
| `fishdata/worldState.json` | 每日鱼讯状态 |

这些运行时数据文件会在首次使用时自动生成，不跟随仓库发布，避免把现有玩家数据一起提交到 GitHub。
更新插件前建议备份 `fishdata/*.json`。

## 常见问题

### 安装后没有反应

先确认插件目录是：

```text
plugins/Fish-plugin
```

然后重启机器人。不要把仓库套成 `plugins/Fish-plugin/Fish-Plugin/Fish.js` 这种多一层目录。

### 怎么切回默认鱼竿和鱼饵

```text
#换竿 0
#换饵 0
```

也可以用：

```text
#换竿 默认
#换饵 默认
```

### 鱼饵为什么会减少

非默认鱼饵是消耗品。装备后每次钓鱼会消耗库存，用完会自动回到清水团饵。

### 特殊鱼竿怎么买

特殊鱼竿不能直接购买，需要钓到对应的 `legendary` 鱼，并把它放在鱼缸里执行 `#炼竿`。
