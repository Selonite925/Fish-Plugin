# Fish-plugin

适用于 TRSS-Yunzai / Yunzai-Bot 的独立钓鱼插件。

体验、反馈QQ群：1103772796

Fish-plugin 提供一套完整的群聊钓鱼玩法：每日钓鱼、鱼缸收藏、鱼市售鱼、鱼蛋商店、鱼竿鱼饵、传说鱼炼竿、每日鱼讯、钓鱼成就和图片面板。

## 功能概览

- `#钓鱼`：每日抛竿钓鱼，有概率钓到不同稀有度的鱼。
- `#钓鱼极速版`：一次结算当前所有可用钓鱼次数，跳过演出并用图片汇总结果。
- `#查看鱼缸`：查看收藏的鱼，支持升级鱼缸容量。
- `#鱼市`：出售鱼获换鱼蛋，购买鱼饵、鱼竿和额外钓鱼券。
- `#鱼竿 / #换竿`：切换不同鱼竿，影响等待时间、上鱼率、失败保护和稀有鱼倾向。
- `#鱼饵 / #换饵`：切换不同鱼饵，鱼饵会在钓鱼时消耗。
- `#炼竿`：消耗鱼缸里的 `legendary` 鱼，炼成特殊鱼竿。
- `#钓鱼祈愿`：100 鱼蛋祈愿 1 次，可获得祈愿限定鱼竿、鱼饵和免费祈愿。
- `#限时鱼讯`：每天刷新高活跃鱼，钓中可获得额外鱼蛋。
- `#钓鱼成就`：查看成就和奖励。
- `#钓鱼管理`：查看管理、维护和主人命令。

## 安装

进入你的 Yunzai-Bot 或 TRSS-Yunzai 根目录，然后执行：

```bash
git clone --depth=1 https://github.com/Selonite925/Fish-Plugin.git ./plugins/Fish-plugin
```

拉取完成后重启机器人。

通常不需要额外安装依赖，插件会使用 Yunzai 已有的运行环境和 puppeteer 图片渲染能力。

### 可选依赖

插件不安装额外依赖也能运行。需要增强时间处理或兼容老 Node 的网络请求兜底时，可以在插件目录执行：

```bash
cd ./plugins/Fish-plugin
pnpm install
```

当前可选依赖包括 `luxon` 和 `node-fetch`。没装时会自动回退，不影响日切、鱼讯、等待时间和普通面板发送。

## 更新

进入机器人根目录，执行：

```bash
git -C ./plugins/Fish-plugin pull
```

更新后重启机器人。

如果服务器直连 GitHub 不稳定，主人可以在机器人内为 `#钓鱼更新` 设置代理：

```text
#钓鱼更新代理
#钓鱼更新代理 https://gh-proxy.com/
#钓鱼更新代理 https://gh-proxy.com/{url}
#钓鱼更新代理关闭
```

未设置代理时仍然使用 GitHub 直连；设置后 `#钓鱼更新` 的 fetch 和安全重装都会通过代理地址拉取。

## 卸载

停止机器人后删除插件目录：

```bash
rm -rf ./plugins/Fish-plugin
```

Windows PowerShell 可用：

```powershell
Remove-Item -Recurse -Force .\plugins\Fish-plugin
```

### 鱼市和售鱼

| 命令 | 说明 |
| --- | --- |
| `#鱼市` | 查看商店、鱼饵、鱼竿和售鱼提示 |
| `#售鱼 1` | 卖出 `#今日鱼获` 里显示的第 1 条 |
| `#售鱼 common` | 卖出今日所有 common 鱼 |
| `#售鱼 common 3` | 卖出今日 common 列表第 3 条 |
| `#售鱼 鱼缸3` | 卖出鱼缸展示列表第 3 条 |
| `#售鱼 全部` | 卖出今日所有可售鱼 |

`legendary` 和彩蛋鱼 `？` 不能出售。

`#售鱼 1` 这类“今日鱼获序号”按 `#今日鱼获` 展示出来的编号来认，不会再额外重排。

售鱼价格会按鱼的稀有度、长度和重量综合计算：越稀有、体型越漂亮，通常越值钱；每个稀有度都有自己的价格区间，避免同类鱼价格浮动过大。`legendary` 和彩蛋鱼 `？` 不能出售，建议优先收藏或用于特殊玩法。

### 买东西

| 命令 | 说明 |
| --- | --- |
| `#鱼市购买 鱼饵1` | 购买鱼市里的第 1 种鱼饵 |
| `#鱼市购买 鱼饵1*5` | 批量购买 5 包鱼饵 |
| `#鱼市购买 鱼饵1 5包` | 同样是批量购买 5 包鱼饵 |
| `#鱼市购买 鱼竿1` | 购买鱼市里的第 1 根鱼竿 |
| `#鱼市购买 额外钓鱼券` | 购买额外钓鱼次数 |
| `#鱼市购买 自定义鱼饵 桂花酒糟` | 用文本定制一份鱼饵 |

鱼饵价格按“包”计算；每包会提供 `packSize` 次库存，例如 1 包 3 份的鱼饵可以在钓鱼时消耗 3 次。

额外钓鱼券只在基础次数用完后消耗，并单独记录为额外竿；鱼缸升级后新增的基础次数仍会按基础竿补给玩家。

### 炼竿

| 命令 | 说明 |
| --- | --- |
| `#炼竿预览 3` | 预览鱼缸展示列表第 3 条鱼会炼成什么鱼竿 |

### 钓鱼祈愿

| 命令 | 说明 |
| --- | --- |
| `#钓鱼祈愿` | 花费 100 鱼蛋祈愿 1 次 |
| `#钓鱼祈愿10` | 连续祈愿 10 次，单次命令最多 10 连 |
| `#钓鱼十连` | 同样是 10 连写法 |
| `#钓鱼祈愿清单` | 查看祈愿清单、当前限定愿品和可能获得的普通愿品 |
| `#钓鱼祈愿概率` | 查看愿池内容、金线进度和各愿品概率 |

当前限定愿品接口挂载 `金满而谦虚之竿`，不能在鱼市直接购买。玩家获得当前限定愿品后，界面会提示“当前限定愿品已获得”或“留些机会给没遇见它的钓友吧”，后续祈愿只会遇见普通愿品。

祈愿获得的鱼饵会明确区分“包”和“份”：祈愿限定鱼饵按包展示，实际进入背包后按份消耗。例如 `渊金鱼饵 1包` 和 `冥潮逆饵 1包` 都是 1 包 1 份，通常可钓 1 次；`谦光素饵 1包` 和 `镜水留饵 1包` 是 1 包 2 份，通常可钓 2 次。`冥潮逆饵` 是代价较重、专门追深层稀有鱼影的祈愿限定鱼饵，更适合收藏或冲图鉴时偶尔使用。`再祈一愿` 和 `免费十连` 会进入免费祈愿次数，结算时优先消耗。

`金满而谦虚之竿` 可以用 `#金谦指定 鱼名` 或 `#金谦目标 rare 金龙鱼` 指定目标鱼：钓到目标稀有度时，目标鱼在同稀有度里的出现感会大大增加，也会有同稀有度鱼影意外闯入。命中真正目标会给鱼蛋奖励，但其它额外鱼蛋收益不会叠加；每天最多调整 1 次目标，重复指定同一目标可刷新闯入鱼影。

### 鱼缸升级

鱼缸每升 1 级增加 5 格容量，并让玩家本人每日基础钓数永久 +2。升级消耗会随等级递增：升到第 N 级需要 N 条 `legendary` 鱼，或 3N 条 `epic` 鱼。

### 管理命令

| 命令 | 说明 |
| --- | --- |
| `#钓鱼管理` | 查看管理命令 |
| `#封竿` | 主人在当前群停用本插件 |
| `#同步鱼缸` | 同步和修正鱼缸数据 |
| `#修复鱼数据` | 修复用户数据结构 |
| `#补鱼 @某人 rare 鳗鱼 80 3.5` | 主人补发鱼获 |
| `#鱼蛋补偿 @某人 100` | 主人补发鱼蛋 |
| `#设置钓鱼刷新时间 6` | 主人设置每日钓鱼日刷新时间，24小时制 |

每日刷新时间默认是 `0:00`。分段式返还默认关闭；开启后，基础次数会在刷新点返还 1/3，刷新点后 6 小时再返还 1/3，刷新点后 12 小时返还剩余全部次数，刷新点后 16 小时起才会使用额外钓鱼券。`#钓鱼极速版` 仍可提前消耗当天全部基础次数，用来减少同一时段刷屏。

## 玩法说明

### 稀有度

鱼的稀有度从低到高大致为：

```text
common -> uncommon -> rare -> epic -> legendary -> ？
```

`？` 是彩蛋稀有度，出现率非常低。

### 鱼竿怎么选

- 想钓得快：优先买 `疾风短竿`、`柳影细竿`。
- 想少空竿：优先买 `稳钓重竿`。
- 想追稀有鱼：优先买 `猎珍长竿`。
- 想守大鱼：优先买 `阔鳞量竿`、`坠岩守竿`。

### 鱼饵怎么选

- 想稳定上鱼：买 `香谷鱼饵`。
- 想多钓普通鱼攒鱼蛋：买 `银鳞鱼饵` 或 `青苔鱼饵`。
- 想追稀有鱼：买 `沉流鱼饵`、`乌贼鱼饵`、`雷息鱼饵`。
- 想玩随机效果：买 `自定义鱼饵`。

| 文件 | 说明 |

| `fishdata/fishData.json` | 用户主数据、鱼缸、鱼蛋、鱼竿、鱼饵、成就 |
| `fishdata/fishConfig.json` | 基础配置、封竿群 |

## 常见问题

### 安装后没有反应

先确认插件目录是：

```text
plugins/Fish-plugin
```

然后重启机器人。不要把仓库套成 `plugins/Fish-plugin/Fish-Plugin/Fish.js` 这种多一层目录。

### 怎么切回默认鱼竿和鱼饵

```text
#换竿 0
#换饵 0
```

也可以用：

```text
#换竿 默认
#换饵 默认
```

### 鱼饵为什么会减少

非默认鱼饵是消耗品。装备后每次钓鱼会消耗库存。自动续饵关闭时，用完会回到清水团饵；开启后会按持有鱼饵列表顺序续到下一种，全部用完后再回到清水团饵。

### 特殊鱼竿怎么买

特殊鱼竿不能直接购买，需要钓到对应的 `legendary` 鱼，并把它放在鱼缸里执行 `#炼竿`。
